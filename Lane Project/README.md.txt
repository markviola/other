#Lane Project #1

A webpage that allows the user to select an image and perform various operations to it such as rotation, translation, shrinking, and changing the opacity. 

Buttons:
* **Rotate:** Rotate the image by 45 degrees
* **Translate:** Translate the image 40px to the left
* **Scale:** Half the size of the image
* **Opacity:** Half the opacity of the image
* **Reset:** Reset the image to its original blank state
